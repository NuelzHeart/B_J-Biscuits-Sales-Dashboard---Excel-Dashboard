# B & J Biscuits Sales Dashboard - Excel Dashboard
## Project Overview
This project aims to design and develop an interactive, visually compelling Sales & Customer Insights Dashboard that delivers key business intelligence for decision-making. The dashboard will consolidate revenue, profitability, customer behavior, and geographic performance into one user-friendly interface, enabling management to quickly assess business performance and optimize strategies.

## Project Objectives
The main goals of the dashboard include:
- Delivering real-time, data-driven insights to support quick and informed decision-making.
- Identifying profitable customer segments, brands, locations, and demographic groups to guide targeted marketing and sales strategies.
- Monitoring key performance indicators (KPIs) such as total revenue, total profit, total quantity sold, and total cost of goods sold.
- Providing interactive filtering to help users drill down into performance by location, payment method, and age group.

## Dashboard Requirements

### Design Preferences
- Clean, interactive, and visually appealing layout
